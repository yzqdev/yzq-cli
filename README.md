# yzq-cli

<a href='https://www.npmjs.com/package/yzq-cli'><img src='https://img.shields.io/npm/v/yzq-cli'/></a>
> A small tool, containing many features.

## install

```shell
pnpm add -g yzq-cli
//or
npm i -g yzq-cli
//or 
yarn global add yzq-cli
```

## Usage

```text
➜ yzq -h
Usage: yzq <command> [options]

Options:
  -v, --version  当前版本.
  -h, --help     如何使用

Commands:
  help           查看帮助
  word           获取诗词
  file           文件相关操作
```
